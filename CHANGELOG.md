# Changelog

## 1.0.0 — 2026-07-16

- подготовлена публичная структура проекта;
- добавлена конфигурация через `.env`;
- добавлена смена пола ребёнка через `ACTUAL_GENDER`;
- секретный путь администратора подключён к маршруту;
- административные API защищены локальной сессией;
- убрана привязка Windows-скрипта к конкретной папке пользователя;
- добавлены переносимые скрипты установки и запуска;
- добавлена валидация `questions.json` и JPG-файлов;
- удаление устаревших вопросов синхронизировано с JSON;
- добавлен нейтральный демонстрационный набор без личных данных;
- добавлены тесты и GitHub Actions для Python 3.11–3.13;
- обновлены зависимости до актуальных совместимых версий;
- добавлены README, инструкция публикации, MIT License и `.gitignore`.
