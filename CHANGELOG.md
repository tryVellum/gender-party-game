# Changelog

## 1.2.1 — 2026-07-22

- исправлено определение корня проекта в PyInstaller spec;
- добавлена предварительная проверка путей сборки;
- исправлена автоматическая Windows-сборка установщика и portable-версии;
- исправлена кодировка пользовательской документации.

## 1.2.0 — 2026-07-21

- добавлен отдельный редактор вопросов и настроек игры;
- добавлено редактирование вариантов и допустимых текстовых ответов;
- добавлены загрузка и удаление JPG/JPEG-изображений;
- добавлены выбор аукционного вопроса и настройка пола ребёнка;
- добавлен синхронный показ правильного ответа после закрытия вопроса;
- исправлено закрытие вопроса после длительного простоя;
- добавлен Windows-лаунчер без консольного окна;
- добавлена защита от второго экземпляра приложения;
- добавлен значок программы в области уведомлений;
- добавлены установщик Windows и portable-версия;
- пользовательские данные перенесены из Program Files в LocalAppData;
- добавлена автоматическая Windows-сборка через GitHub Actions;
- браузерный Socket.IO-клиент включается в готовую сборку;
- добавлены тесты путей первого запуска и полный smoke-тест;
- обновлена пользовательская инструкция и правила игры.

## 1.0.0 — 2026-07-16

- подготовлена публичная структура проекта;
- добавлена конфигурация через `.env`;
- добавлена смена пола ребёнка через `ACTUAL_GENDER`;
- секретный путь администратора подключён к маршруту;
- административные API защищены локальной сессией;
- убрана привязка Windows-скрипта к конкретной папке пользователя;
- добавлены переносимые скрипты установки и запуска;
- добавлена валидация `questions.json` и JPG-файлов;
- удаление устаревших вопросов синхронизировано с JSON;
- добавлены Ruff, Pytest и GitHub Actions для Python 3.11–3.13.
