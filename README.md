# Gender Party Game

[![Tests](https://github.com/tryVellum/gender-party-game/actions/workflows/ci.yml/badge.svg)](https://github.com/tryVellum/gender-party-game/actions/workflows/ci.yml)

[![Latest release](https://img.shields.io/github/v/release/tryVellum/gender-party-game)](https://github.com/tryVellum/gender-party-game/releases/latest)

Локальная многопользовательская викторина для гендер-пати. Один компьютер используется как экран ведущего, а гости отвечают со смартфонов через общую Wi‑Fi-сеть.

## Скачать игру

Откройте страницу:

**[Скачать последнюю версию](https://github.com/tryVellum/gender-party-game/releases/latest)**

В разделе **Assets** выберите один из файлов:

- `GenderPartyGame-Setup-1.2.1.exe` — рекомендуемый установщик для Windows;
- `GenderPartyGame-Portable-1.2.1.zip` — переносная версия без установки.

Не скачивайте `Source code (zip)` для обычной игры: это исходный код для разработчиков.

## Установка на Windows

1. Скачайте `GenderPartyGame-Setup-1.2.1.exe`.
2. Запустите файл.
3. Оставьте включённым пункт **«Создать ярлык на рабочем столе»**.
4. Завершите установку.
5. Запустите ярлык **Gender Party Game**.

Программа устанавливается в:

```text
C:\Program Files\Gender Party Game
```

Изменяемые вопросы, изображения, настройки и результаты хранятся отдельно:

```text
%LOCALAPPDATA%\GenderPartyGame
```

Поэтому обновление программы не удаляет пользовательские вопросы и настройки.

### Предупреждение Windows SmartScreen

Установщик пока не подписан платным сертификатом издателя. Windows может показать сообщение **«Система Windows защитила компьютер»**.

Нажмите:

```text
Подробнее → Выполнить в любом случае
```

Скачивайте установщик только из официального раздела **Releases** этого репозитория.

### Запрос брандмауэра

При первом запуске Windows может запросить доступ к сети.

Разрешите доступ для:

```text
Частные сети
```

Доступ для публичных сетей включать не нужно.

## Быстрый старт

Перед игрой компьютер ведущего и телефоны гостей должны быть подключены к одной Wi‑Fi-сети.

1. Запустите **Gender Party Game** с рабочего стола.
2. Дождитесь автоматического открытия страницы администратора.
3. До начала игры откройте **Редактор игры** и проверьте:
   - вопросы;
   - правильные ответы;
   - изображения;
   - аукционный вопрос;
   - правильный пол ребёнка.
4. Вернитесь на экран игры.
5. Нажмите кнопку с QR-кодом.
6. Покажите QR-код гостям.
7. Каждый игрок сканирует код и вводит уникальный никнейм.
8. Когда все подключились, ведущий открывает первый вопрос.

После скачивания и установки интернет не требуется. Нужна только общая локальная Wi‑Fi-сеть.

## Как закрыть программу

Закрытие вкладки браузера не останавливает сервер игры.

Чтобы полностью завершить работу:

1. найдите значок **Gender Party Game** рядом с часами Windows;
2. нажмите по нему правой кнопкой;
3. выберите **«Завершить игру»**.

Через это же меню можно:

- открыть экран администратора;
- открыть редактор вопросов;
- открыть папку пользовательских данных.

## Редактор игры

Редактор открывается с экрана администратора или через значок программы рядом с часами.

В редакторе можно:

- изменить текст всех 20 вопросов;
- изменить варианты ответа;
- отметить один или несколько правильных вариантов;
- перечислить допустимые текстовые ответы;
- загрузить или удалить JPG/JPEG-изображение;
- выбрать один аукционный вопрос или отключить аукцион;
- задать правильный пол ребёнка.

Для текстового вопроса допустимые ответы можно вводить через запятую или с новой строки. Например:

```text
плацента, детское место
```

Ограничения изображений:

- формат JPG/JPEG;
- размер до 5 МБ;
- не более 20 мегапикселей.

Редактирование доступно только до начала игры. Если вопрос уже был открыт, сначала нажмите **«Сбросить игру»**.

## Правила игры

### Игровое поле

Поле состоит из четырёх тем. В каждой теме пять вопросов стоимостью:

```text
100, 200, 300, 400 и 500 баллов
```

Вопросы бывают:

- с выбором варианта;
- с вводом текста;
- с изображением;
- аукционные.

### Обычный вопрос

1. Ведущий нажимает карточку вопроса.
2. Вопрос появляется у всех игроков.
3. Игрок выбирает вариант или вводит текст.
4. Ответ можно изменить, пока ведущий не закрыл вопрос.
5. Ведущий повторно нажимает на карточку и фиксирует ответы.

Начисление баллов:

- правильный ответ: `+стоимость вопроса`;
- неправильный ответ: `−стоимость вопроса`;
- ответ не отправлен: `0`.

После фиксации правильный ответ показывается:

- игрокам — около 10 секунд;
- ведущему — около 5 секунд.

Затем игра возвращается к игровому полю.

### Аукционный вопрос

В аукционе участвуют только игроки, которые:

- находятся на связи;
- имеют больше 0 баллов на момент начала аукциона.

Порядок:

1. Ведущий открывает аукционную карточку.
2. Каждый участник делает одну закрытую ставку.
3. Ставка не может быть отрицательной или превышать текущий рейтинг игрока.
4. После ставок всех участников определяется победитель.
5. Побеждает максимальная ставка.
6. При одинаковых ставках выигрывает тот, кто отправил ставку раньше.
7. Сам вопрос показывается только победителю аукциона.
8. После ответа ведущий закрывает карточку.

Начисление:

- правильный ответ: `+ставка`;
- неправильный ответ: `−ставка`;
- ответ не отправлен: `0`.

Игроки с нулевым или отрицательным рейтингом ждут завершения аукциона и не делают ставку.

### Финальный раунд

Финальный раунд становится доступен после закрытия всех 20 вопросов.

1. Ведущий нажимает **«Финальный раунд»**.
2. Каждый игрок выбирает:
   - **Мальчик**;
   - **Девочка**.
3. Выбор можно менять до раскрытия.
4. Ведущий нажимает на финальную карточку.
5. Запускается барабанная дробь.
6. Пол ребёнка одновременно раскрывается на всех устройствах.

Игрокам, которые угадали, удваивается текущий рейтинг:

```text
новый рейтинг = текущий рейтинг × 2
```

Неверный выбор не изменяет рейтинг.

Обратите внимание: отрицательный рейтинг также умножается на два.

### Секретный раунд имён

После раскрытия пола ведущий нажимает на финальную карточку ещё раз.

1. Каждый игрок может предложить только одно имя.
2. Все предложенные имена появляются в общем списке.
3. Игроки могут отдавать свои баллы понравившимся именам.
4. Переданные баллы вычитаются из рейтинга игрока и добавляются рейтингу имени.
5. Имена сортируются от большего рейтинга к меньшему.

### Победители

В игре можно определить двух победителей:

- игрок с наибольшим количеством баллов;
- имя с наибольшим рейтингом в секретном раунде.

## Полный сброс

Кнопка **«Сбросить игру»** полностью начинает новую игру и удаляет:

- игроков;
- баллы;
- ответы;
- ставки;
- отметки использованных вопросов;
- финальные голоса;
- предложенные имена и их рейтинг.

Тексты вопросов, изображения и установленный пол ребёнка при этом сохраняются.

## Portable-версия

1. Скачайте `GenderPartyGame-Portable-1.2.1.zip`.
2. Распакуйте архив в отдельную папку.
3. Запустите `GenderPartyGame.exe`.

Не запускайте EXE прямо из ZIP-архива.

Данные portable-версии хранятся рядом с программой:

```text
user-data
```

Перед заменой portable-версии сохраните папку `user-data`, чтобы не потерять вопросы и настройки.

## Резервная копия вопросов

Откройте меню значка программы рядом с часами и выберите **«Папка данных»**.

Для полной резервной копии сохраните папки:

```text
data
instance
```

В них находятся вопросы, изображения, настройки игры и база результатов.

## Частые проблемы

### Телефон не открывает игру

Проверьте:

- телефон и компьютер подключены к одной Wi‑Fi-сети;
- на телефоне отключён VPN;
- телефон не использует только мобильный интернет;
- сеть не является гостевой сетью с изоляцией устройств;
- в брандмауэре Windows разрешён доступ для частных сетей;
- компьютер не перешёл в спящий режим.

### QR-код открывает неправильный адрес

Перезапустите программу после подключения компьютера к нужной Wi‑Fi-сети и снова откройте QR-код.

### В браузере написано «Соединение потеряно»

- убедитесь, что программа продолжает работать в области уведомлений;
- обновите страницу;
- повторно запустите ярлык — второй сервер не создастся, откроется уже работающая игра.

### Не воспроизводится звук

Нажмите один раз в любом месте экрана администратора. Современные браузеры могут блокировать звук до первого действия пользователя.

### Не запускается программа

Откройте:

```text
%LOCALAPPDATA%\GenderPartyGame\logs\gender-party-game.log
```

В файле будет записана причина ошибки.

### Порт 5000 занят

Закройте ранее запущенную копию программы через значок рядом с часами. Если значка нет, перезагрузите компьютер и запустите игру снова.

## Безопасность и конфиденциальность

Игра предназначена для доверенной домашней или офисной локальной сети.

Не нужно:

- открывать порт игры в интернет;
- настраивать проброс порта на роутере;
- публиковать личные фотографии и вопросы без согласия изображённых людей.

Перед публичным релизом замените семейные вопросы и изображения на нейтральный демонстрационный набор.

- рекомендуется Python 3.12.

```powershell

```

    runtime_paths.py version.py build/generate_version_info.py tests
