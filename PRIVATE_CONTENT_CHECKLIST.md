# Проверка личных данных перед публикацией

Перед каждым публичным `git push` проверьте:

- [ ] в `data/questions.json` нет фамилий, адресов и личных историй;
- [ ] в `data` нет семейных фотографий, которые нельзя публиковать;
- [ ] файл `.env` не отслеживается Git;
- [ ] база `instance/game.sqlite` не отслеживается Git;
- [ ] в проекте нет паролей, токенов и приватных ключей;
- [ ] у всех добавленных изображений есть право на публичное распространение;
- [ ] фактические вопросы проверены на корректность;
- [ ] тесты `pytest -q` проходят.
